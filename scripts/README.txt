This directory contains example scripts that performance individual directive command operations.
  * alloc_rsc.sh  - allocate stream resource
  * disable_str.sh - disable stream directive 
  * get_dir_param.sh/identify_dir.sh - get supported directives' parameters  
  * get_str_status.sh - get current stream directive status 
  * release_strid.sh - release a stream ID
  * str_rw.sh - stream write and compare data
  * enable_str.sh - enable stream directive  
  * get_str_param.sh - get stream directive parameters 
  * release_rsc.sh - release all allocated stream resources
  * str_write.sh - stream write with specified stream ID
  * sf_multistream.job - FIO configuration that performs writes in 4 streams.
